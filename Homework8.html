Lesson 8: Supplemental: Polynomial and Rational Inequalities

Inequalities

Given a function \(y=f(x)\) the following inequalities are often considered.

  • \(f(x)>0\), \(y\) is greater than zero. Graphically, this is the interval(s) on the \(x\)-axis where the function is above the \(x\)-axis.

  • \(f(x) \geq 0\), \(y\) is greater than or equal to zero. Graphically, this is the interval(s) on the \(x\)-axis where the function is above the \(x\)-axis or on the \(x\)-axis.

  • \(f(x)<0\), \(y\) is less than zero. Graphically, this is the interval(s) on the \(x\)-axis where the function is below the \(x\)-axis.

  • \(f(x) \leq 0\), \(y\) is less than or equal to zero. Graphically, this is the interval(s) on the \(x\)-axis where the function is below the \(x\)-axis or on the \(x\)-axis.


\(x\)-intercept of a function \(y=f(x)\) is real number \(c\) such that

\(f(c)=0\).


A function is undefined when there is not a valid output for a given

input. A function \(y=f(x)\) is undefined at a real number \(c\) if: \(f(c)\) results in division by zero; taking the square(or even root) of a negative number; or later evaluating a logarithm at a non-positive number.


A Boundary Point of a function \(y=f(x)\) is a real number \(c\) for which

\(f(c)=0\) or \(f(c)\) is undefined. Boundary points might be referred to as cut points.


Solving Polynomial Inequalities

  1. Rewrite so that one side of the inequality is zero.

  2. Find all boundary points. Usually, factoring is the appropriate method.

  3. Create a sign chart or table around the boundary point(s) to partition the real number line into intervals.

  4. Evaluate the inequality at a point in each interval(don’t use the endpoints) and the result will either be true or false.

  5. The solution will be the union of all true intervals.


For examples 1-2, solve the inequality and write the solution as an interval.

Example 1:

\(x^2-9x+20 \le 0\)





Example 2:

\(-x^3 \le x^2-132x\)





Solving Rational Inequalities

  1. Rewrite so that one side of the inequality is zero.

  2. Find all boundary points. Usually, factoring is the appropriate method. Find the zeros of the numerator and the denominator.

  3. Create a sign chart or table around the boundary point(s) to partition the real number line into intervals.

  4. Evaluate the inequality at a point in each interval(don’t use the endpoints) and the result will either be true or false.

  5. The solution will be the union of all true intervals.


For examples 3-4, solve the inequality and write the solution as an interval.

Example 3:

\(\dfrac{x-4}{x+2} \geq 0\)





Example 4:

\(\dfrac{(x-9)(x+9)}{x} \le 0\)

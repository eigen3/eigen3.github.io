Section 1.3: Rates of Change and Behavior of Graphs

The Average Rate of Change of a function \(f(x)\) defined on \([a,b]\) is defined as:

\[m=\dfrac{\text{change in y}}{\text{change in x}}=\dfrac{\Delta y}{\Delta x}=\dfrac{f(b)-f(a)}{b-a}=\dfrac{y_2-y_1}{x_2-x_1}\]

Example 1: Find the average rate of change of \(f(x)=x^2+\sqrt{x}-\frac{1}{x}\) on the interval \([4,9]\).

Intervals of Increasing, Decreasing, and Constant Let \(f(x)\) be a function defined on an interval \(I\) and let \(a,b \in I\) and suppose \(a<b\).

  • If \(f(a)<f(b)\), then \(f(x)\) is increasing on the interval I.

  • If \(f(a)>f(b)\), then \(f(x)\) is decreasing on the interval I.

  • If \(f(a)=f(b)\), then \(f(x)\) is constant on the interval I.

Extrema:

A function \(f(x)\) has a local maximum at a point \(b\) if in an open interval \((a,c)\), \(f(b) \ge f(x)\) for all \(x \in (a,c)\). A function \(f(x)\) has a global or absolute maximum at a point \(b\) if \(f(b) \ge f(x)\) for all \(x\) in the domain of \(f\).

A function \(f(x)\) has a local minimum at a point \(b\) if in an open interval \((a,c)\), \(f(b) \le f(x)\) for all \(x \in (a,c)\). A function \(f(x)\) has a global or absolute minimum at a point \(b\) if \(f(b) \le f(x)\) for all \(x\) in the domain of \(f\).

Concavity or curvature of a graph

A function \(f(x)\) is concave upward on an interval \(I\) if its rate of change is increasing.

A function \(f(x)\) is concave downward on an interval \(I\) if its rate of change is decreasing.

A point of inflection \((c,f(c))\) is a point on the graph of \(f\) where the function changes concavity.

Section 1.4: Composition of Functions

\[(f \circ g)(x)=f(g(x))\]

This is read \(f\) composed with \(g\). This composition is valid for all \(x\) in the domain of \(g\) for which \(g(x)\) is in the domain of \(f(x)\).

Domain of a Composition: To find the domain of a composition \(f(g(x))\), find the intersection of the domain of \(g(x)\) and \(f(g(x))\).

Guidelines for solving rational inequalities:

  • Factor the numerator and the denominator.

  • Find the boundary points(BP). A boundary point is a zero of the numerator or the denominator.

  • Set up a sign chart around your boundary points and test in each interval.

  • The solution will be the union of true intervals.

Lesson 11: Section 3.3: Intervals of Increasing, Decreasing and

Constant. Extrema

Let \(y=f(x)\) be a function defined on an interval \(I\).

  • \(f(x)\) is on \(I\) if for each \(x_1 , x_2 \in I\) such that if \(x_1<x_2\), then \(f(x_1)<f(x_2)\).





  • \(f(x)\) is on \(I\) if for each \(x_1, x_2 \in I\) such that if \(x_1<x_2\), then \(f(x_1) > f(x_2)\)





  • \(f(x)\) is on \(I\) if for each \(x_1 , x_2 \in I\) such that if \(x_1<x_2\), then \(f(x_1)=f(x_2)\).

Extrema is plura for minimum and maximum of a function.

Let \(y=f(x)\) be a function defined on an interval \(I\).

  • Local Minimum: \(f(a)\) is a local minimum of \(f(x)\) on \(I\) if there exists a subinterval \(J\) of \(I\) such that \(a \in J\) and \(f(a) \le f(x)\) for all \(x \in J\).

  • Local Maximum: \(f(a)\) is a local maximum of \(f(x)\) on \(I\) if there exists a subinterval \(J\) of \(I\) such that \(a \in J\) and \(f(a) \geq f(x)\) for all \(x \in J\).

  • Global Minimum:

    \(f(a)\) is a global(or absolute) minimum of \(f(x)\) on \(I\) if \(f(a) \leq f(x)\) for all \(x \in I\).

  • Global Maximum: \(f(a)\) is a global(or absolute) maximum of \(f(x)\) on \(I\) if \(f(a) \geq f(x)\) for all \(x \in I\).

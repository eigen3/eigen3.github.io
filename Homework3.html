Lesson 3: Section 2.2: Linear Equations in One Variable

To solve a linear equation, isolate the variable on one side of the equation.



Example 1:

\(-2-2(x+4)=2x+1\)





Example 2:

\(12(x-7)=28x-256\)





Example 3:

\(\dfrac{8}{x}+\dfrac{2}{x}=12\)





Example 4:

\(\dfrac{x-2}{5}+5=\dfrac{x-1}{4}\)





Example 5:

\((x-2)(x+4)=(x+3)^2\)





Example 6:

\(\dfrac{4x+2}{5x+9}=\dfrac{4x+8}{5x+7}\)





Solving Literal Equations, isolate the variable to be solved.



Example 7: Solve for \(h\).

\(V=\dfrac{1}{3}bh\)





Example 8: Solve for \(r\).

\(S=\dfrac{a}{1-r}\)

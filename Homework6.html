Lesson 6: Sections 2.6: Radical Equations and Equations in Quadratic Form

Radical Equations

  1. Isolate a radical.

  2. If there is a square root, raise both sides of the equation to the second power. If there is a cube root, raise both sides of the equation to the third power. If there is an nth root, raise both sides of the equation to the nth power.

  3. If there is still a radical, repeat step 1, otherwise solve the resulting linear or quadratic equation.

  4. Check the solution(s).


For examples 1-5, solve the radical equation and check to the solution(s).


Example 1:

\(\sqrt{x-7}=5\)





Example 2:

\(\sqrt{x+4}=-2\)





Example 3:

\(\sqrt{3x+18}=x\)





Example 4:

\(\sqrt{x+149}=x+17\)





Example 5:

\(\sqrt{x}+3=\sqrt{x+12}\)





Equations in Quadratic Form: \(au^2+bu+c\)


  1. Identify the expression to be replaced by \(u\) and check that \(u^2\) replaces the remaining variable expression in terms of \(u\).

  2. Factor and solve in terms of \(u\).

  3. Replace \(u\) with the expression it replaced and solve. Check solutions.


For examples 6-9, solve use the substitution method.

Example 6:

\(2(x+3)^2+11(x+3)+15=0\)





Example 7:

\(x^4+x^2-20=0\)





Example 8:

\(\left(\dfrac{x}{x+1}\right)^2 +\dfrac{20x}{x+1}=-99\)





Example 9:

\(x^{-2}-8x^{-1}+15=0\)

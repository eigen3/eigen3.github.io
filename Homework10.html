The Domain of a Function \(y=f(x)\) is the set of all real numbers such

that \(f(x)\) is defined (\(\text{denominator}\ne 0\)) and real (\(\text{radicand} \ge 0\)).


To find the domain of a function:

  • The default is \((-\infty, \infty)\) unless

  • There is a denominator, find the restrictions, \(\text{denominator} \ne 0\).

  • There is a radical of even degree, usually it will be square root, for \(\sqrt{\text{radicand}}\), find the restrictions, \(\text{radicand} \ge 0\)

Piece-wise defined functions are functions that are defined using two

or more functions. A simple example of a piece-wise defined function would be a mobile phone bill. For instance, a flat rate is charged for using up to 10gb of data and once this has been exceeded, a charge is applied per gigabyte used.

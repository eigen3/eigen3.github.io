Lesson 1: Section 1-1: Functions and Section 1.2: Domain and Range

Summer 2023

Definitions:

  • A relation is a set of ordered pairs, usually \((x,y)\).

  • The set of all input values(first coordinate(\(x\))) of a relation is the domain.

  • The set of all output values(second coordinate(\(y\))) of a relation is the range.

  • A function is a relation where each input(\(x\)) has exactly one output(\(y\)).(no \(x\) repeats) \(a=b\) implies \(f(a)=f(b)\).

Example 1: Determine whether the following relation is a function. Find the domain and the range. \[\{ (0,30), (1,30), (2,30), (3,30), (4,70), (5,70), (6,70) \}\]

Example 2: Graph the following function. Find the domain and the range. Sketch the graph.

\[f(x)= \begin{cases} 30 & \text{if } 0 \le x < 3 \\ 70 & \text{if } 3 \le x <10\\ 2(x-10)+70 & \text{if } x \ge 10\\ \end{cases}\]

Example 3: Determine whether the following relation is a function, find the domain and range. Graph. \[\{(4,2), (1,1), (0,0), (1,-1), (4,-2) \}\]

The Vertical Line Test The graph of a function will be intersected at most once by any vertical line.

One-to-One Function A one-to-one function(1-1 or invertible) is a function where each output has exactly one input. (no \(y\) nor \(x\) repeats). \(f(a)=f(b) \text{ if and only if } a=b\).

Example 4: Determine whether the function is 1-1.

(a) \(f(x)=2x+5\)

(b) \(f(x)=\mid x \mid = \begin{cases} -x & \text{if } x<0 \\ x & \text{if } x \ge 0 \\ \end{cases}\)

The Horizontal Line Test A function is one-to-one if any horizontal line intersects the graph of the function at most once.

Absolute Value Inequalities For \(b>0\) , \(|x|<b\) is equivalent to \(-b<x<b\).

For \(b>0\), \(|x|>b\) is equivalent to \(x<-b\) or \(x>b\).

Example 7: Solve the inequality and write the solution as an interval. \[-2|x+3|-1 \le -25\]

Example 9: Write the equivalent inequality and sketch the equivalent graph representing \[(-\infty,1] \cup (3,7]\]

Domain of a function \(f(x)\) The domain of a function \(f(x)\) is the set of all real numbers for which the function is real and defined. Procedure for finding the domain:

  • The default domain is \((-\infty, \infty)\) unless

  • denominator \(\ne 0\)

  • \(\sqrt[n]{\text{radicand}}\), where \(n\) is even radicand \(\ge 0\).

  • \(\log_b(\text{argument})\) , argument \({}>0\).

Example 10: Find the domain of the function \(f(x)=\dfrac{\sqrt{5-x}}{x^2+11x+10}\) and write in interval notation.

The Range of a function \(f(x)\) is the set of all outputs.

Graphing piece-wise defined functions. Always use the point(s) where the function transfers from one piece to the next. These point(s) where the function transfers are the only point(s) that will possibly be open. All other points will be closed.

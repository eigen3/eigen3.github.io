Relation and Functions

  • A relation is a set of ordered pairs. (Usually \((x,y)\)).

  • The Domain of a relation is the set of all \(x\) values.

  • The Range of a relation is the set of all \(y\) values.

  • A Function is a relation where each input \(x\) has exactly one output \(y\).(In other words, no \(x\) repeats).

  • A one-to-one Function is a function where each output has exactly one input. (In other words, no \(x\) nor \(y\) can repeat). One-to-one is often called “1-1” or “invertible”.

  • The \(x\) is the input(independent variable)

  • The \(y\) is the output(dependent variable) of a function.

Graphs of Relations

  • The domain of a graph of a relation is read from left to right and written as an interval.

  • The range of a graph of a relation is read from bottom to top and written as an interval.

  • The Vertical Line Test states that the graph of a function will be intersected by any vertical line at most once.

  • The Horizontal Line Test states that the graph of a one-to-one function will be intersected by any horizontal line at most once.

Function notation: \(y=f(x)\) communicates that \(y\) is a function of

\(x\). \(x\) is the independent variable and \(y=f(x)\) is the dependent variable. As an ordered pair this is \((x,f(x))\)


To evaluate a function at \(c\) replace all \(x\) with \((c)\) and then follow the order of operations. If the input was a real number, the output is a real number provided that the expression is defined. If the input was an expression of \(x\), for example, then the output will be an expression of \(x\).
